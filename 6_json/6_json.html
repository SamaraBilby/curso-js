JSON - JavaScript Object Notation

  • Utilizado para comunicação entre serviços, ex: back end <-> front end
  • Um tipo de dado padronizado, que lembra os objetos do JS
  • Rigoroso na estrutura, ou seja, deve seguir o padrão imposto por este formato;
  • Apenas aspas duplas
  • Não aceita comentários

JSON para String

  • Podemos converter o JSON para uma string de forma fácil ou de string para JSON
