O objeto do evento

  • Quando criamos eventos, temos a opção de utilizar um argumento opcional, que é chamdao de objeto do evento
  • Ele contém propriedades que podem ser utilizadas a nosso favor
  • O objeto é criado pelo JS automaticamente

    • function msg(e){
    console.log(e);
    }
    btn.addEventListener("click", msg);

Propagação

  • Quando não definimos um elemento muito bem(seletor brando) ou um elemento que está dentro de outro tem um evento, pode acontecer a Propagação, ou seja, o outro elemento ativar um evento também, aí teremos uma duplicação;
  • Para isso temo um método que para esta Propagação e resolve este problema, o StopPropagation;
