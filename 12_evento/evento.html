O objeto do evento

  • Quando criamos eventos, temos a opção de utilizar um argumento opcional, que é chamdao de objeto do evento
  • Ele contém propriedades que podem ser utilizadas a nosso favor
  • O objeto é criado pelo JS automaticamente

    • function msg(e){
    console.log(e);
    }
    btn.addEventListener("click", msg);

Propagação

  • Quando não definimos um elemento muito bem(seletor brando) ou um elemento que está dentro de outro tem um evento, pode acontecer a Propagação, ou seja, o outro elemento ativar um evento também, aí teremos uma duplicação;
  • Para isso temo um método que para esta Propagação e resolve este problema, o StopPropagation;

Eventos de tecla(Key event)

  • Sempre que um tecla é pressionada, são gerados dois eventos: keyup e keydown;
  • Podemos realizar ações nestes eventos também;
  • keyup é quando soltamos tecla;
  • keydown é quando apertamos;

Outros eventos de mouse

  • No mouse também temos eventos como mousedown e mouseup, semelhante aos das teclas
  • dblclick para ativar com dois cliques

Movimento do mouse

  • Podemos ativar eventos com a movimentação do mouse também
  • o nome dele é mousemove;
  • Através desse evento podemos detectar a posição do ponteiro do mouse na tela.
