Booleans

  • Serve para guardar um valor de uma comparação
  • os únicos valores possíveis são:
  • true
  • false

Booleans - comparações

  • Maior que: >
  • Menor que: <
  • Maior ou igual: >=
  • Menor ou igual: <=
  • igual: ==
  • diferente: !=
  • idêntico: ===

Booleans - Operadores Lógicos

  • && - and -> para ser true, os dois lados da comparação tem que ser verdadeiro
  • || - or -> para ser true, basta um dos lados da comparação ser verdadeiro
  • ! - not -> inverter valores, ex: o que é true vira falso

Booleans - operador ternário

  • Faz um comparativo em apenas uma linha de cógigo
  • Ex: console.log(true ?1:2) console.log(false ?'falso':'verdadeiro')
  • Não é muito utilizado e pode deixar o código complicado de ler
