Callbacks

  • Callback é uma função que faz uma ação após algum acontencimento no código
  • Podemos realizar um callback com a função setTimeout

Promises

  • São ações assíncronas que podem produzir um valor algum momento no código
  • Uma forma de dizer a linguagem que um valor pode estar presente em um futuro do código
  • O objeto das promises é Promise, resolve é o método que resolve uma Promise, o then é o que faz ela poder ser executada em um ponto futuro

Falha nas Promises

  • Uma Promise pode falhar, podemos reter esse erro com um método chamado catch;
  • Com ele podemos exibir erro no console, por exemplo, e fazer o debug no código;

Rejeitando Promises

  • Além do resolve, há o método reject
  • Que é quando determinada lógica não satisfa nosso pragrama, então podemos ir para outra com o reject, em vez do resolve
  • Resolve e reject terminam a Promise, ou seja, não podemos chamar mais o then.

Resolvendo várias Promises

  • Com o método all, podemos resolver várias promessas de uma vez só
  • Ou seja passamos elas pelo array e quando a última for resolvida, receberemos a resposta;

Async Functions

  • Podemos criar funções assíncronas com a palavra reservada async;
  • Elas retornam um Promise;
  • Se retornar algo, a promessa é resolvida,se der alguma exception a promessa é rejeitada;
